# Installation

[![Get it from NuGet](https://raw.githubusercontent.com/hmlendea/readme-assets/master/badges/stores/nuget.png)](https://nuget.org/packages/NuciCLI)

**.NET CLI**:
```bash
dotnet add package NuciCLI
```

**Package Manager**:
```powershell
Install-Package NuciCLI
```
